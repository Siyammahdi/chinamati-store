# 🏺 Chinamati

A modern e-commerce platform designed to provide a seamless shopping experience for customers while showcasing products with a clean and elegant interface.

## Features

- Responsive design across all devices
- Product listing and details pages
- Product search and filtering
- Shopping cart functionality
- Secure authentication
- Order management
- Fast and optimized performance
- Modern UI/UX

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Supabase

### Deployment
- Vercel

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/chinamati.git
cd chinamati
````

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Start Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Run Production Build

```bash
npm start
```

## Deployment

Deploy easily using Vercel.
